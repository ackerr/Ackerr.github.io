Golang实现git-open

ZmJ
2020/09/20
cli, git, go

git-open是GitHub上开源的一个git相关命令行工具，终端通过键入git open，会在浏览器中打开当前路径所在git项目的远端页面，支持多种代码托管平台，具体可查看官方文档。本质上git-open是一个shell脚本，核心代码仅有200+行。在无参数的前提下，主要有以下几步

  1. 判断当前路径是否在git工作区内
  2. 获取当前branch的remote-url
  3. 转换remote-url为托管仓库的页面地址
  4. 根据不同平台，命令行唤起浏览器

下文将依照以上步骤，通过Golang第三方库cobra来实现类似命令行功能，具体cobra用法，可移步官方文档

完整代码放置文章结尾，git open功能也已集成至ackerr/lab。
